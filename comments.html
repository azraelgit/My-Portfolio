Overall Structure

My website consists of four HTML documents (index.html, past.html, present.html, comments.html) as content and one CSS file (websystems.css) to define the styling. Each page is consistent with styling and structure and has the header and menu navigation combined, appearing at the top across all webpages. As such, each page is accessible through the menu navigation, allowing the user to explore other parts of the website freely.

Internal Structure

Each HTML document contains elements or tags to format parts of the document.

  • .header - Contains logo and navigation menu
  • .nav - Navigation menu in all pages
  • .a href - Provide access to other pages of the website
  • .section - Groups content
  • .div - Groups content further
  • .h1 - .h4 - Heading heirarchy
  • .p - Groups text in paragraphs
  • .img - Show image with alt text for accessibility
  • .br and .hr - Divide content for spacing
  • .footer - Page credits

Aesthetics & Accesibility

I went for a minimilastic design, with a background-colour of white as my primary style, for my website to strip away unnecessary clutter and present myself and my content in a approachable manner. As such, i've incorporated a select few font colours for styling, that passes the WebAIM Accesibility Contrast Checker making the text accessible to read, including dark gray (#3A3C40, Contrast ratio 11.05:1) for my pargraphs and navigation text, and select h1 headings and navigation hovers with purple (#50309C, Contrast ratio 9.35:1) to contrast with the white-space in my website.

I've chosen purple as the unique colour to give life to my minimalistic website, as it's my favourite colour, to ensure certain words within headings pop out more and make it so the reader notices it first. Also, in certain areas of my webpages, I use a dark-gray background within a .div or .section to contrast the white space and give the website more character and definition. To accompany it, instead of using dark-gray text, I use white text (#FFFFFF) to contrast the background, and is seen within this text box. All texts uses a font-size of 17px and a line-height of 1.5 for further accesiblity in mind, making it easier for the viewer to read, and a web safe font (Verdana, sans-serif) and font-size of for readability. Alt text is also used to describe the contents of an .img. This ensures screen readers that are visually impaired can understand the content. It also serves as a fallback whenever an image fails to load.

Spacing and Interactivity

I use consistent spacing for my .padding and .margin to construct a uniform structure across my website. I also use various sizes of .gap and .margin-top/.margin-bottom to polish certain areas and present a clean style. Some elements such as .nav (in header) and .img (in index.html) use effects to give visual feedback to the user that they are clickable or important.

Spacing of Elements

  • Used 25px for most padding and margins.
  • Employed various gap, margin-top, and margin-bottom values to maintain a minimalistic structure.

Interactivity with navigation links and images

  • All navigation links include a hover effect (underline and colour shift to purple #50309C).
  • Certain images (.img within index.html) have a zoom effect (transform: scale(1.05)).
